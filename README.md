### Tecnologias usadas

- React JS
- Styled-components
- Javascript
- HTML
- CSS

### Features

- Tela de apresentação exibindo todas as cartas com seu conteúdo visível, e com um botão para iniciar o jogo.
- Ao clicar no botão, as cartas são viradas - escondendo o conteúdo - e embaralhadas.
- Permite que o usuário selecione apenas uma carta, clicando na mesma
- Apresenta a carta selecionada, o nome da carta e uma descrição.
